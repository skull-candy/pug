# RFC1628 UPS-MIB Mapping

Project repository: `https://git.vns.ae/ahsan/pug`

PUG implements a small read-only RFC1628 UPS-MIB subset:

| OID | Name |
| --- | --- |
| `1.3.6.1.2.1.33.1.1.1.0` | manufacturer |
| `1.3.6.1.2.1.33.1.1.2.0` | model |
| `1.3.6.1.2.1.33.1.1.3.0` | firmware |
| `1.3.6.1.2.1.33.1.1.4.0` | agent software |
| `1.3.6.1.2.1.33.1.1.5.0` | name |
| `1.3.6.1.2.1.33.1.2.1.0` | battery status enum |
| `1.3.6.1.2.1.33.1.2.2.0` | seconds on battery |
| `1.3.6.1.2.1.33.1.2.3.0` | estimated minutes remaining |
| `1.3.6.1.2.1.33.1.2.4.0` | charge remaining percent |
| `1.3.6.1.2.1.33.1.4.1.0` | output source enum |
| `1.3.6.1.2.1.33.1.4.4.1.2.1` | output voltage |
| `1.3.6.1.2.1.33.1.4.4.1.5.1` | output load percent |
